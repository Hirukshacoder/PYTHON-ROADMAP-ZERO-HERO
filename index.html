III - Operators

In Python there are mathematical symbols like:- + [addition], - [substraction], / [division], * [multiplication].

However, there are some other operators as well: "=" is used to assign a variable, "==" is mostly used in conditions to express something equals to another, "+=" refers to add a value to the same value/variable, "-=" refers to substract from value to the same variable, "*=" refers to times a value to the same variable, "/=" refers to divide a value from the same variable.

  • " % " this sign refers to mod
  • " // " this sign will divide the first number with the second and round it to the nearest integer.
Example: 

                                        
            x = 2 # variable 
            x += 1 
            print(x)
            # output 3 

            y = 2 # variable 
            y -= 1
            print(y)
            # output 1

Feel free to try out the other operators! Assign variables using `=` sign; For the record variables are empty `data-locations` until we assign a value for them.
